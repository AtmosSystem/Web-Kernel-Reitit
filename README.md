# Atmos Web Kernel for [Reitit](https://github.com/metosin/reitit)

This library is the web core of atmos systems, it has commons functionality using *Reitit*.

## Installation
```clojure
[atmos-web-kernel-reitit "x.x.x"] ;See clojars version
```
